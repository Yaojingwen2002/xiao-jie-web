科技 2小时前

OpenAI 发布 GPT-5 预览版，展示新一代 AI 能力

阿邮简介：OpenAI 发布了 GPT-5 技术预览，展示了对复杂推理、长上下文理解的新突破...

详细报道：

OpenAI 今日发布了 GPT-5 的技术预览版，这是继 GPT-4 之后的又一重大更新。新版本在多个关键领域实现了突破：

  • 复杂推理能力：在数学、物理等科学问题上的准确率提升 40%
  • 长上下文理解：支持最高 200万 token 的上下文窗口
  • 多模态进化：图像、视频、音频的理解和生成能力大幅增强
  • 实时学习：能够在对话中动态学习用户偏好

Sam Altman 表示，GPT-5 将在今年晚些时候正式发布，企业版 API 将优先开放。
